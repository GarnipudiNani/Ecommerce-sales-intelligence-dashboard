# E-Commerce Sales Intelligence Dashboard

## Dashboard Preview

![Dashboard](dashboard_overview.png)

## Project Overview

This Power BI dashboard analyzes E-Commerce sales performance from January 2023 to December 2024.

### Key Metrics

- Total Revenue: ₹16.96M
- Total Orders: 5,000
- Average Order Value (AOV): ₹3.39K
- Profit Margin: 38.57%
- Total Profit: ₹6.54M

## Features

- Revenue & Orders Trend Analysis
- Regional Revenue Distribution
- Product Category Performance
- Top Products by Revenue
- Interactive Year and Region Filters

## Tools Used

- Power BI
- Excel
- DAX
- Data Visualization

## Files Included

- Ecommerce_Dashboard.pbix
- ECommerce_Clean_Dataset_2026.xlsx
- PowerBI_Dashboard_ECommerce_2026_RealData.html

## Business Insights

1. Electronics generated the highest revenue.
2. East region contributed the largest revenue share.
3. Tablets and Speakers were top-performing products.
4. Revenue remained stable across both years.
